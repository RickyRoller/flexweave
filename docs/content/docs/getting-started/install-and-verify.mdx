---
title: Install and Verify
description: Add Flexweave to a Rust runtime and verify the local workspace.
---

Add Flexweave as a dependency from the published crate once a release is
available:

```toml
[dependencies]
flexweave = "0.1"
```

Inside this workspace, verify the crate directly:

```bash
cargo test -p flexweave
cargo run -p flexweave --example rpg_combat
```

Run the complete crate and documentation gate from the workspace root:

```bash
bun run verify
```

## First Import

Most runtimes start with object identity and attached data:

```rust
use flexweave::{DataStore, ObjectId, ObjectStore};

let mut objects = ObjectStore::new();
let player: ObjectId = objects.create();

let mut names = DataStore::new();
names.attach(player, "Aria");
```

The next page explains why this ordinary Rust code stays in your runtime rather
than being hidden behind a Flexweave framework object.
