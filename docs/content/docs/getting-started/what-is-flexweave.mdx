---
title: What Flexweave Is
description: Flexweave provides deterministic mechanics primitives for Rust.
---

Flexweave is a Rust crate for domain-agnostic mechanics state. It gives you
object ids, attached data, attributes, tags, abilities, active effects, clocks,
lifecycle facts, event channels, signals, and cleanup helpers.

## Main Example

The guide builds from one compile-checked example:

```bash
cargo run -p flexweave --example rpg_combat
```

Read [Create Combatants](/docs/rpg-combat/01-create-combatants) when you are
ready to start building it piece by piece.
